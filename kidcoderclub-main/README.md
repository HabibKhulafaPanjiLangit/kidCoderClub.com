
# KidCoderClub Project

## Informasi Proyek

Website ini adalah platform pembelajaran pemrograman untuk anak dan remaja. Proyek ini dikembangkan menggunakan teknologi modern agar mudah digunakan dan dikembangkan.

## Cara Edit Kode

Anda dapat mengedit kode dengan beberapa cara:

### 1. Menggunakan IDE Favorit

Clone repository ini dan lakukan perubahan secara lokal. Setelah selesai, push perubahan ke repository.

**Langkah-langkah:**
```sh
git clone <URL_REPOSITORY_ANDA>
cd <NAMA_FOLDER_PROYEK>
npm install
npm run dev
```

### 2. Edit Langsung di GitHub

Anda dapat mengedit file langsung di GitHub dengan fitur edit dan commit.

### 3. Menggunakan GitHub Codespaces

Jalankan Codespaces untuk mengedit dan menjalankan proyek langsung di cloud.

## Teknologi yang Digunakan

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Cara Deploy

Deploy dapat dilakukan dengan menghosting aplikasi ini di layanan hosting statis seperti Vercel, Netlify, atau GitHub Pages. Pastikan untuk membangun proyek dengan perintah:
```sh
npm run build
```
Ikuti panduan masing-masing layanan hosting untuk proses deploy.

## Custom Domain

Anda dapat menghubungkan domain kustom melalui pengaturan layanan hosting yang digunakan. Ikuti dokumentasi resmi layanan hosting untuk mengatur domain.
